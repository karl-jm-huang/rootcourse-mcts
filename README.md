# rootcause

## 文件描述：

- hotspot.py: 二维版本代码

- version1.py: 单线程版本，可进行五个维度的根因分析，需要较长时间。

- version2.py: 多线程版本，可进行五个维度的根因分析。

- run.py: 用于执行整个文件夹数据。

- test_data: 数据集

- record_test1.md: 数据集描述

**tips**: version1.py中，元素值使用下标表示，若集合为(0, *, 5, *, *, *)， 则对应seq为(0, -1, 5, -1, -1,- 1)
